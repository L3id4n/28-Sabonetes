# AGRODECIDE

## Problema

Pequenos produtores frequentemente têm dificuldade em avaliar a rentabilidade de suas culturas e identificar oportunidades de melhoria.

## Solução

Uma ferramenta simples que centraliza informações da safra e gera indicadores para apoiar a tomada de decisão.

## Público-alvo

Pequenos e médios produtores rurais.

## Benefício

- Menos decisões por "achismo".
- Melhor controle financeiro.
- Identificação de culturas mais rentáveis.
